# 🤖 Общий бот для задач и расходов

Бот для двух пользователей — шеф и ты видите общие задачи и расходы.

---

## 🚀 Запуск за 5 минут (бесплатно через Railway)

### Шаг 1 — Создай бота в Telegram

1. Открой Telegram, найди **@BotFather**
2. Напиши `/newbot`
3. Придумай название (например: `Наши задачи`)
4. Придумай username (например: `nashizadachi_bot`)
5. BotFather пришлёт **токен** — скопируй его, он выглядит так:
   ```
   7412345678:AAHxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
   ```

---

### Шаг 2 — Запусти на Railway (бесплатно)

1. Зайди на **railway.app** и войди через GitHub
2. Нажми **New Project → Deploy from GitHub repo**
3. Загрузи три файла: `bot.py`, `requirements.txt`, `Procfile`
4. В настройках проекта перейди в **Variables** и добавь:
   - `BOT_TOKEN` = твой токен от BotFather
5. Railway сам установит зависимости и запустит бота

> 💡 Если не хочешь Railway — можно запустить локально (см. ниже)

---

### Шаг 3 — Локальный запуск (альтернатива)

```bash
pip install -r requirements.txt
BOT_TOKEN="твой_токен" python bot.py
```

Или вставь токен прямо в `bot.py` в строке:
```python
BOT_TOKEN = "ВСТАВЬ_ТОКЕН_СЮДА"
```

---

## 📱 Как пользоваться

1. Оба пользователя открывают бота в Telegram и пишут `/start`
2. Шеф добавляет задачу → ты открываешь бота → видишь её сразу
3. Ты добавляешь расход → шеф видит его у себя
4. Все данные общие — в файле `data.json` на сервере

---

## ✨ Возможности

- 📋 **Задачи** — добавить, отметить выполненной, список
- 💰 **Расходы** — добавить с суммой, удалить, итого
- 👤 Видно кто добавил и когда
- 🔄 Общие данные для обоих пользователей
